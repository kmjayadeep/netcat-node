# Netcat - Node
Netcat Equivalent built using node.js and sockets

### Requirements

1. Node.js
2. NPM

## Installation

* Clone the Repo

* Install dependencies


```
npm install
```

* Start server

redirect output to stdout

```
npm start
```

or redirect output to a file

```
npm start > output.txt
```


* Start client 

Input from stdin
```
node client.js
```

or Input from a file

```
node client.js < input.txt
```
